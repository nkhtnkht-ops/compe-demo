予約チェック管理かんたん使い方

コンペ予約のフォロー連絡を、やり漏れなく進めるためのアプリです。

アプリを開く 毎日の使い方を見る

Chrome または Edge で開いてください。編集はすべて自動で保存されます。

Start

Daily Operation

毎日の使い方

画面の赤いまるのところを、上から順に押していくだけです。下の番号と画面を見ながら進めてください。

（朝いちばん）「再接続する」が出たら1回押す

朝アプリを開くと、上に黄色い帯や 権限再付与 ボタンが出ることがあります。これは前回の台帳ファイルにつなぎ直すだけの操作です。1回押せばいつも通り使えます。出ていなければ、そのまま次へ進んでOKです。

Operations

よく使う操作

「こんな時に使う」→「やり方」の順で書いています。必要になった時だけ見れば大丈夫です。

連絡の結果を付ける

こんな時：幹事へ連絡したあと、結果を残したい時。

やり方：予約に 〇／不在／不要 を選びます。すぐには消えず、「更新（整理）」を押すまで一覧に残ります。

名前でさがす

こんな時：特定の予約をすぐ見つけたい時。

やり方：上の検索ボックスに名前を入れると、その予約に移動します。

連絡タイミングで絞り込む

こんな時：「受付の翌日ぶんだけ」「35日前ぶんだけ」など、同じ回をまとめて連絡したい時。

やり方：「全部／受付翌日／60日前／35日前／18日前」から選ぶと、その回の予約だけが並びます。

急ぎかどうかを見分ける

こんな時：すぐ連絡すべきか、後でよいかを知りたい時。

やり方：予約に付く色つきの札（バッジ）で分かります。

遅延2日本日あと3日繰越

不在のとき、次に電話する日を決める

こんな時：つながらなかった予約を、指定した日にまた出したい時。

やり方：不在の予約に「次に電話する日」を設定します。その日まで一覧から外れ、当日にまた出てきます。

④まで一気に飛ばす

こんな時：メンバーや定例コンペなど、前半の連絡がいらない時。

やり方：①〜③をまとめて「不要」にして、④だけ残します。

元に戻す

こんな時：直前に付けた結果を取り消したい時。

やり方：「元に戻す」を押すと、1つずつ取り消せます（「更新（整理）」を押すまで有効）。

⟳ 更新（整理）

こんな時：済んだ予約を今日の一覧から外したい時。

やり方：「⟳ 更新（整理）」を押すと、済んだ予約が「今日やること」から外れます。

Excelで控えを出す

こんな時：今の台帳をExcelの控えとして残したい時。

やり方：「Excel書き出し」を押すと、Excelファイルで保存できます。あとで取り込み直すこともできます。

SMS List

SMSの配信リストを作る

台帳から「この人たちに送りたい」を絞り込んで、SMS配信サービス（SMSLINK）に取り込める名簿を書き出します。たとえば「外部予約サイト経由の幹事さんに、次回は公式予約をご案内」といった使い方ができます。

① 送る相手を絞る

  • 予約の経路（外部予約サイト／自社予約・電話）で選びます。
  • プレー日の期間、キャンセル除外、未来の予約の除外。
  • 「○回以上来た幹事だけ」など、回数でも絞れます。
  • 海外のお客様を外す設定もできます。
  • 携帯番号（070/080/090）がある人だけが対象です。

② 名簿に入る項目

Excelで開ける名簿ファイルとして書き出します。

携帯番号氏名姓最終プレー日コース経路予約回数

③ 書き出す手順

  • 上の「SMS配信リスト」を開く。
  • 条件を選び「件数を確認」で何人になるか見る。
  • 「SMSLINK用に書き出し」で保存。

「○○様」と名前を入れて送る

  • SMSLINKに名簿をアップロードし、「携帯番号」の列を選びます。
  • 本文を作るとき「姓」を選ぶと、一人ひとりの名前が自動で入ります。
  • 例：「中村様 いつもご利用ありがとうございます…」のように届きます。

同じ人は1件にまとめます

  • 同じ携帯番号は1件にまとめ、いちばん新しいプレー日を使います。
  • まとめた回数は「予約回数」に入るので、常連の幹事さんが分かります。
送る前に確認を
これらの連絡先は、予約・お客様対応のためにお預かりしたものです。販促のSMSに使うときは、目的やお客様の同意の範囲を確認してから配信してください。

毎日の始め方・困ったとき

  • 朝アプリを開いて「台帳ファイルを選んでください」と出たら、中央のボタンから共有フォルダの台帳ファイルを選びます（前回のファイル名が出ます）。
  • 「権限再付与」ボタンが出たら、1回押すだけで前回の台帳につながります。
  • PCによっては毎朝選び直しが必要なことがあります（故障ではありません。データは消えていません）。
  • みんなで同じ共有フォルダの台帳を使えます。上のファイル名で、正しい台帳につながっているか確認しましょう。

「未接続」と出たら

台帳ファイルにまだつながっていない、という意味です。あわてず、中央のボタンから共有フォルダの台帳ファイルを選び直してください。それだけでつながります。

状態の例未接続権限再付与保存済み

自動バックアップ（最初に1回だけ設定）

設定しておくと、保存のたびに日付つきの控えが共有フォルダへ自動で残ります。万一データが消えても、ここから戻せます。

  • 設定の手順は、上の「設定」ページの「自動バックアップの設定」を見てください（番号つきで説明しています）。
  • 一度設定すれば、あとは保存のたびに自動。直近2週間分の控えがたまります。
設定の手順を見る →

もしデータが消えたら

  • 自動バックアップを設定していれば、共有フォルダの日付つきの控えから戻せます（操作が不安なときは管理担当へ連絡を）。
  • 控えが無くても、Excel台帳から取り込み直せば元に戻せます。
  • ふだんから「Excelで控えを出す」も併用すると安心です。
ネットに預けていません
お客様の氏名・電話番号などは、インターネット上のサーバーには保存しません。共有フォルダの台帳ファイルだけにあります。
控えは2種類
「自動バックアップ」で日付つきの控えが自動で残ります。さらに「Excelで控えを出す」でExcelの控えも作れます。
